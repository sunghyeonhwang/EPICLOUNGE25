# 작업 리포트 (2024-01-08)

## 1. 메인 페이지 및 슬라이드 애니메이션 최적화

- **GSAP SplitText 적용**: 메인 슬라이드의 제목과 텍스트에 SplitText 애니메이션을 적용하여 단어/글자 단위의 역동적인 효과 구현.
- **깜빡임 방지 개선**: CSS(visibility: hidden, opacity: 0)와 JS(beforeChange 이벤트)를 결합하여 슬라이드 전환 시 텍스트가 깜빡이는 현상 완벽 제거.
- **슬라이드 루프 대응**: Slick 슬라이더의 `beforeChange`와 `afterChange` 이벤트를 정교하게 제어하여 슬라이드가 루프될 때도 항상 애니메이션이 정상적으로 시작되도록 최적화.

## 2. 디자인 고도화 및 UI/UX 개선

- **리소스 섹션**: 타이틀 폰트 크기를 **18px**로 상향하여 가독성 강화.
- **'전체보기' 버튼**: 크기를 축소하고 세련된 회색 톤(#e5e5e5)으로 변경하여 보조 메뉴로서의 역할 최적화.
- **새소식 이미지 박스**: 곡률을 **24px**로 조정하고, 호버 시 **콘트라스트 강화 및 부드러운 확대** 효과를 추가하여 생동감 부여.
- **검색창 디자인**:
  - 검색 상자에 **20px 둥근 모서리** 적용.
  - 클릭 시 나타나는 파란색 **outline 하이라이트 제거**.
  - 관리 편의를 위해 전용 스타일 파일(`common26.css`)로 분리 관리.
- **로고 교체**: 상단 로고를 새로운 버전(`logo_new.svg`)으로 교체.

## 3. 관리자 페이지(ADM) 기능 개선

- **로그아웃 버튼**: 우측 상단의 '라운지 관리'를 **'로그아웃'**으로 변경.
- **리다이렉트 설정**: 로그아웃 버튼 클릭 시 로그아웃 처리 후 즉시 관리자 로그인 페이지(**`/v3/adm`**)로 이동하도록 로직 수정.
- **메뉴 통합**: '배너 통합 관리' 메뉴 신설 및 관련 파일 관리 효율화.

## 4. 구조 설계 변경

- **스타일 시트 분리**: 기존 `common.css`에 영향을 주지 않도록 `common26.css`를 신규 생성하여 독립적인 디자인 적용이 가능하도록 구조화.
